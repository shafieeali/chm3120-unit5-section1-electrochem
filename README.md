# Electrochemistry and Potentiometry Introduction

Static GitHub Pages app for `Unit5_Section1_intro to electrochem`.

Built using `Lecture_App_Master_Prompt_and_Workflow.docx` as the governing workflow.

## Includes

- Lecture-sequence tabs based on slide ranges.
- Topic-specific interactive lab: Electrochemical Cell Builder, Nernst Equation Concentration Lab, Reference Electrode Coach.
- Slide Studio with 54 slide images and accessible transcripts.
- Original lecture PDF download.
- Practice cards extracted from instructor-provided practice materials: 74 cards.
- No Word files exposed in the student-facing app.
- Keyboard navigation, visible focus, live feedback, text-first mode, graph summaries, and data tables.
